## Kebutuhan Bisnis
- Web Aplikasi digunakan untuk menghitung balistik
- Web aplikasi terdapat beberapa fitur dan menu
- Web aplikasi memiliki halaman login dan registrasi
- Web aplikasi memiliki beberapa menu :
  1. Dashboard
  2. Calculator Balistik
  3. User Management (Profile)
  4. Calculator Zero Point
- untuk menu Calculator Balistik terdapat landing page yang berisi table list data dengan filter by date, kemudian ada fitur untuk create calcultion yang berupa form isian.
- untuk dashboard berisi summary aktivitas calculator
- untuk user management berisi pengaturan data pengguna seperti email no hp dan password
- untuk menu Calculator Zero Point terdapat landing page yang berisi table list data dengan filter by date, kemudian ada fitur untuk create calcultion yang berupa form isian.

## Menu Calculator Balistik
- Berisi Landing page yang berisi list data hasil kalkulasi
- Form Create terdiri dari beberapa tab :
    1. DATA UMUM TKP
       - pada tab ini terdapat beberapa input : 
        - [ ] Tanggal & waktu ditemukan
        - [ ] Lokasi presisi TKP
        - [ ] Kondisi cuaca
        - [ ] Kondisi pencahayaan
        - [ ] TKP indoor/outdoor (di dalam atau luar ruangan)
        - [ ] Status TKP masih asli atau sudah berubah
        - [ ] Dokumentasi foto 360°
        - [ ] Video TKP sebelum barang dipindah

    2. POSISI KORBAN
        - pada tab ini terdapat beberapa section yaitu :
            - Identifikasi posisi tubuh
                - pada section ini terdapat input :
                    - [ ] Posisi tubuh saat ditemukan
                    - [ ] Posisi kepala
                    - [ ] Posisi tangan
                    - [ ] Posisi kaki
                    - [ ] Posisi duduk/berdiri/terlentang
                    - [ ] Posisi korban terhadap benda sekitar (kursi, pohon, kendaraan, dll)
            - Pengukuran posisi
                - pada section ini terdapat input : 
                    - [ ] Tinggi luka masuk dari tanah
                    - [ ] Tinggi luka keluar dari tanah (jika ada)
                    - [ ] Tinggi posisi proyektil di dalam tubuh
                    - [ ] Sudut tubuh korban
                    - [ ] Arah hadap korban
            - Kondisi biologis
                - pada section ini terdapat input :
                    - [ ] Pola darah
                    - [ ] Blood spatter (percikan darah akibat benturan/tembakan)
                    - [ ] Pooling darah (genangan darah karena korban diam di satu posisi)
                    - [ ] Tanda perlawanan
                    - [ ] Livor mortis (pengendapan darah setelah mati)
                    - [ ] Rigor mortis (kaku mayat)
    3. PEMERIKSAAN LUKA TEMBAK
    4. PEMERIKSAAN PAKAIAN
    5. BARANG BUKTI BALISTIK
    6. ANALISA TRAJECTORY (LINTASAN PELURU)
    7. RESIDU MESIU / GSR
    8. PEMERIKSAAN LAB BALISTIK
    9. PARAMETER KRUSIAL UNTUK REKONSTRUKSI
    10. TARGET AKHIR PENYELIDIKAN
